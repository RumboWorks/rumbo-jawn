# Agent Specialists

This directory contains specialist instructions for maintaining specific Rumbo docs or reviewing specific work areas.

These files are instructions, not source-of-truth project content.

Examples:

- `roadmap.agent.md` explains how to maintain `docs/active-planning/roadmap.md`.
- `decision-log.agent.md` explains how to maintain `docs/active-planning/decision-log.md`.
- `usage.agent.md` explains how to maintain `docs/reference/usage.md`.
- `phase-review.agent.md` explains how to perform phase closeout reviews.

Coding agents should not read every specialist file by default. Read the specialist file only when the current task calls for that role.
