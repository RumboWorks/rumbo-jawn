# Planning Pack Manifest

This manifest describes the files included in the Rumbo planning pack.

## Root files

- `README.md` — repository overview and reading path.
- `AGENTS.md` — global coding-agent guidance and documentation hygiene rules.
- `MANIFEST.md` — this file.

## Documentation

- `docs/README.md` — documentation map and reading guide.
- `docs/project-charter/` — stable project agreements.
- `docs/tools/` — tool-specific product notes and scope boundaries.
- `docs/active-planning/` — roadmap, decision log, deferred work, retrospectives, implementation notes.
- `docs/development-phases/` — phase-by-phase execution guidance.
- `docs/reference/` — verified operational/reference documentation.
- `docs/working-notes/` — temporary or exploratory notes.
- `docs/archive/` — superseded material.

## Agent guidance

- `.agent/` — provider-neutral specialist playbooks.
- `.github/` — GitHub Copilot guidance adapters.
- `CLAUDE.md` — Claude Code guidance adapter.

## Important distinction

`docs/active-planning/decision-log.md` is the actual decision log.

`.agent/decision-log.agent.md` is the instruction file for maintaining the decision log.
