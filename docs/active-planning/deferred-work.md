# Deferred Work

Nothing should be postponed without being captured here or moved to a named future phase.

Use this format:

```md
## Deferred item: Short title

Originally from: Phase N — Name  
Deferred to: Phase N or Later / Backlog  
Reason: Why this is not being completed now.  
Blocking: What must happen before this can resume.  
Must not forget because: Why this matters.  
Status: Open / Closed
```
