# Working Notes

Use this directory for temporary research, implementation observations, and agent-created notes.

Naming convention:

```text
YYYY-MM-DD-short-topic.md
```

Working notes are not source of truth. During phase closeout, review any working notes created during the phase and either:

- promote useful content into source-of-truth docs,
- link the note from a current planning/phase doc,
- move it to `docs/archive/`, or
- delete it if it was temporary.
