# Phase 05 — Sounds Like Us Guidance Workbench

## Purpose

Build the dynamic Sounds Like Us guidance output screen where users can adjust, combine, and export guidance options.

## What this phase delivers

- Guidance workbench UI.
- Option toggles/sliders where appropriate.
- Output variants derived from internal structured output where possible.
- Download/copy outputs as appropriate.
- Feedback/rating collection if in scope.
- React island/page if vanilla JS is no longer appropriate.

## Out of scope

- Model Eval features.
- Full paid prompt-piece editing unless roadmap is revised.
- Final visual design.
- Embeddable widgets unless moved earlier.

## Acceptance criteria

- User can view generated guidance.
- User can adjust output options.
- Output updates are understandable and reliable.
- Repeat AI calls are avoided when existing structured data is sufficient.
- UI approach remains compatible with Twig shell and shared SCSS.
